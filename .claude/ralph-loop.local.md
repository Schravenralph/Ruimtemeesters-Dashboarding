Build a full Primos-style dashboarding platform mirroring primos.abfresearch.nl. Use the Ruimtemeesters-Geoportaal folder structure as the project scaffold. Features: theme-based dashboard pages for Bevolking, Huishoudens, Woningen, Woningtekort. Tile-based configurable grid layout. Chart components including bar, stacked bar, line, pie, radar, table, choropleth map. Geographic filtering with gemeente/regio/wijk search. Period/year selection with comparison mode. RBAC and ABAC access control. Custom dashboard creation with shareable URLs. Export as PDF, Excel, CSV, PNG per tile. PostgreSQL data layer with Dutch municipality seed data. Express API with auth middleware. Tech stack: React 19, TypeScript, Vite, Tailwind CSS 4, Express 5, PostgreSQL, Zod. --duration 7.5h
